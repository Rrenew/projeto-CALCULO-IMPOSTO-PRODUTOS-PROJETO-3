# Cálculo Imposto Produtos

Sistema web para cadastro de produtos com cálculo automático de impostos.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Funcionalidades

- Cadastro de produtos (descrição, características, valor, unidade, tipo)
- Listagem dinâmica com campo de quantidade
- Cálculo automático: valor total, imposto e valor final
- Tributação por tipo: 0% | 8% | 10% | 12% | 17%
- Remoção de produtos
- Layout responsivo


## Como executar

Abra o arquivo `index.html` no navegador.

## Regras de negócio

- Campos obrigatórios não podem ficar vazios
- Produtos tipo 1 são isentos de imposto
- Cálculos atualizados em tempo real
- Dados mantidos apenas em memória
